## Contact Management System in C

This is a simple Contact Management System developed in C. It lets users store, manage, and retrieve contact information using a console-based interface. It is aimed at beginners who want to learn about file handling, structures, and basic CRUD operations in C.

### Features

* Add new contacts
* View all saved contacts
* Search contacts by name or phone number
* Update existing contact details
* Delete contacts
* Data stored persistently using file handling

### Technologies Used

* C Programming Language
* File Handling (fopen, fread, fwrite, etc.)
* Structures for organizing contact data

### How to Run

1. Clone the repository.
2. Compile the program using:

   ```bash
   gcc contact_management.c -o contact_management
   ```
3. Run the executable:

   ```bash
   ./contact_management
   ```

### Purpose

This project was created to practice key C concepts such as:

* Structures
* Functions
* File I/O
* Menu-driven programs
